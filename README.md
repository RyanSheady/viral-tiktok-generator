## Environment Variables

Create a `.env.local` file with:

```env
OPENAI_API_KEY=your_api_key_here
OPENAI_MODEL=gpt-4
```

## Development

```bash
npm install
npm run dev
```

## Deployment

1. Push to GitHub
2. Import to Vercel
3. Add environment variables in Vercel dashboard
4. Deploy!
